
# **1. Algebra**

### **Q1. Solve the system of equations**

Solve for (x,y):

[
\begin{cases}
x+2y=5\
3x+4y=11
\end{cases}
]

**Solution:**

Using matrix inverse / elimination:

[
\begin{aligned}
&3x+4y=11,\
&\text{Multiply first eqn by 3: }3x+6y=15,\
&\text{Subtract: }(3x+6y)-(3x+4y)=15-11\Rightarrow 2y=4\Rightarrow y=2,\
&x+2(2)=5\Rightarrow x+4=5\Rightarrow x=1.
\end{aligned}
]

**Answer:** (\boxed{x=1,\ y=2}) ([SATHEE][1])

### **Q3. Two red balls from a bag**

A bag contains 5 red and 3 blue balls. Two balls are drawn at random (without replacement). Find the probability both are red.

**Solution:**

[
P(\text{2 red}) = \frac{\binom{5}{2}}{\binom{8}{2}}=\frac{10}{28}=\frac{5}{14}.
]

**Answer:** (\boxed{5/14}) ([SATHEE][1])

### **Q4. Sum 7 on two dice**

Two fair dice are rolled. Compute (P(\text{sum}=7)).

**Solution:**

Total outcomes = 36; favorable = {(1,6),(2,5),(3,4),(4,3),(5,2),(6,1)} = 6.

[
P=6/36=1/6.
]

**Answer:** (\boxed{1/6}) ([SATHEE][2])

### **Limits**

### **Q5. Evaluate**

[
\lim_{x\to 0}\frac{\sin x - x\cos x}{x^3}
]

**Solution:**

Expand series:

[
\sin x=x-\tfrac{x^3}{6}+\cdots,\quad \cos x=1-\tfrac{x^2}{2}+\cdots,
]
[
\sin x - x\cos x=(x-\tfrac{x^3}{6})-x(1-\tfrac{x^2}{2})=x-\tfrac{x^3}{6}-x+\tfrac{x^3}{2}=\tfrac{x^3}{3}.
]
Thus
[
\lim_{x\to0}\frac{x^3/3}{x^3}=\frac{1}{3}.
]

**Answer:** (\boxed{1/3}) ([SATHEE][3])

### **Optimization (Max/Min)**

### **Q7. Simple max/min**

Find the maximum value of (f(x)=-(x-2)^2+9).

**Solution:**

This is an upside-down parabola with vertex at (x=2):

[
\max f(2)=-(0)^2+9=9.
]

**Answer:** (\boxed{9\text{ at }x=2})

### **Q8. Solve by Cramerâ€™s Rule**

Solve:
[
\begin{cases}
2x+3y=13\
4x-y=5
\end{cases}
]

**Solution:**

[
D=\begin{vmatrix}2&3\4&-1\end{vmatrix}=2(-1)-4(3)=-2-12=-14,

[

]
[
D_y=\begin{vmatrix}2&13\4&5\end{vmatrix}=2(5)-4(13)=10-52=-42.

[

]

**Answer:** (\boxed{x=2,\ y=3})
